宝塔自动挂载数据盘

挂载之前，如果无法挂载或提示非LINUX分区。
可以使用格式化命令
mkfs.ext4 /dev/sdb
命令执行中输入y确认
完毕后即可执行如下命令。
Centos/Fedora：
yum install wget -y && wget -O auto_disk.sh https://git.yds.ink/auto_disk.sh && bash auto_disk.sh
Ubuntu:
wget -O auto_disk.sh https://git.yds.ink/auto_disk.sh && sudo bash auto_disk.sh
Debian:
wget -O auto_disk.sh https://git.yds.ink/auto_disk.sh && bash auto_disk.sh
